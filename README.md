🐾 FurHelp – Smart Pet Care & Appointment System

https://youtu.be/HmfJAYrW_go

Overview

FurHelp is a web application that streamlines pet healthcare. Users can search for veterinarians, book appointments, view prescriptions, and track pet health. Doctors can manage appointments, approve or reject bookings, issue prescriptions, and maintain separate views for current and past consultations.

Key Features

User Portal: Search vets, book appointments, view prescriptions, pet health analysis

Doctor Portal: Approve/reject appointments, issue prescriptions, manage consultation links

Dashboard: Intuitive UI with light/dark mode, session-based authentication

Tech Stack

Backend: Python, Flask

Frontend: HTML, CSS, Bootstrap

Database: SQLite / Custom DB via user2.py

Authentication: Session-based for users and doctors
